# Новшества
## Статьи и комментарии
### С версией 2025.12.0 добавлена функциональность
- Статьи (Post) с CRUD операциями
- Форматированный контент с поддержкой HTML/текста
- Изображения внутри статей с персонализированными путями хранения
- Теги (PostTag) с ManyToMany связью
- Комментарии (PostComment) с поддержкой иерархии (ответы)
- Статусы статей и комментариев
- Автоматические даты создания/обновления/публикации
- Фильтрация через django-filters
- Документирование API через drf-spectacular (Swagger)
- Разделение эндпоинтов по логическим группам
### С версией 2025.12.1 будут расширена документация и добавлены тесты
